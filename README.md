# kafka-connect-toolkit

Custom Kafka Connect transforms and Debezium converters for common data-pipeline tasks.

**Documentation:** [https://nryanov.github.io/kafka-connect-toolkit/](https://nryanov.github.io/kafka-connect-toolkit/)

## Build

```shell
make package
```

See the [install guide](https://nryanov.github.io/kafka-connect-toolkit/getting-started/install/) for deployment steps.
